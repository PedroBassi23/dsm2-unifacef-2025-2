PROVA2 - projeto pronto

Passo a passo para rodar:

1) instale o node (versão 16+ recomendada) e o npm
2) instale expo cli global (opcional):
   npm install -g expo-cli

3) dentro da pasta do projeto rode:
   npm install
   npm start

4) para abrir no navegador:
   no terminal aperte "w" ou abra http://localhost:19002 e clique em "Run in web browser"

5) para simular mobile no navegador:
   abra as ferramentas de desenvolvedor do navegador (F12) e clique no ícone do dispositivo móvel antes de tirar os prints

6) prints:
   salve 3 capturas (cada tela) dentro da pasta prints

7) subir no github:
   git init
   git add .
   git commit -m "prova 2"
   git branch -M main
   git remote add origin <seu repo>
   git push -u origin main

Dependências que este projeto espera:
@react-navigation/native
@react-navigation/native-stack
react-native-screens
react-native-safe-area-context
expo

Se quiser, instale manualmente:
npm install @react-navigation/native @react-navigation/native-stack react-native-screens react-native-safe-area-context
